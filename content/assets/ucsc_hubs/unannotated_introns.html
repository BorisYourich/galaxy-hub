Unannotated Introns

The data from the Annotated Introns track is obtained from the JBrowse instance at plasmodb.org, https://plasmodb.org/plasmo/jbrowse/index.html?data=%2Fa%2Fservice%2Fjbrowse%2Ftracks%2Fdefault&tracks=RNA-Seq%20Evidence%20for%20Introns which is curated by VEuPathDB (https://veupathdb.org, release 38). These data are obtained from the "Unannotated (Strong Evidence)" subtrack of the "RNA-Seq Evidence for Introns" track. Data files of type BigBed are derived from accessing their API to get feature data, which is then converted to GFF3, then to Bed, and finally to BigBed, so understand that there are certainly several places along the way for errors to be introduced. You should always check data you find on this hub against what is available at VEuPathDB.
